# KidSearch-AI Full Package

## Frontend
Upload `frontend/index.html` to GitHub Pages.

## Backend Setup
1. Install Node.js
2. Go to backend folder
3. Run:
   npm install
   node server.js

## Replace YOUR_OPENAI_API_KEY
Get your API key from https://platform.openai.com/

## Hosting Backend
Use Render.com or Railway.app (Free)

## Connect Frontend
Replace YOUR_BACKEND_URL in index.html with your backend URL.
